# homepage

This is my homepage, the latest production build is available [here](https://www.justix.dev).

## Setup

```bash
bun install
```

## Development Server

Development server is available at `http://127.0.0.1:3000`.

```bash
bun --bun run dev
```

## Build for Production

```bash
bun run build
```

Locally preview production build:

```bash
bun run preview
```
